$\displaystyle\lim_{x \to 0}\frac{\left(x+4\right)^{\frac{3}{2}}+e^{x}-9}{x}$

$f\left(x\right)=\left(x+4\right)^{\frac{3}{2}}+e^{x}-9$

$f\left(0\right)=0$

so we have a form of $\frac{0}{0}$

$f'\left(x\right)=e^{x}+\frac{3\sqrt{x+4}}{2}$

we then have $\frac{e^{x}+\frac{3\sqrt{x+4}}{2}}{1}$ by l'hopital's rule

$f'\left(0\right)=4$

so $\displaystyle\lim_{x \to 0}\frac{\left(x+4\right)^{\frac{3}{2}}+e^{x}-9}{x} = 4$

doing some manipulation we can do this

$\ln\left(\left|x\right|\right)\sin\left(x\right)$

$=\frac{x\ln\left(\left|x\right|\right)\sin\left(x\right)}{x}$

$=x\ln\left(\left|x\right|\right)\cdot\frac{\sin\left(x\right)}{x}$

$\displaystyle \lim_{x \to 0}\frac{\sin\left(x\right)}{x} = 1$

so by that we have $x\ln\left(\left|x\right|\right)$
we can then get this in the form $\frac{\ln\left(x\right)}{\frac{1}{x}}$

if we do l'hopital's rule we get $\frac{\left(\frac{1}{x}\right)}{\left(-\frac{1}{x^{2}}\right)}$

witch is -x so $\displaystyle \lim_{x \to 0} \ln\left(\left|x\right|\right)\sin\left(x\right) = 0$
